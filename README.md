# Aplikasi Pengelolaan Asrama Mahasiswa Putri Tarakan Paguntaka
========================================================================
# PABowo
#### Deskripsi Aplikasi
Aplikasi ini merupakan sistem informasi manajemen asrama yang dirancang untuk membantu proses pencatatan iuran dan pengelolaan inventaris secara digital. Aplikasi ini dapat dikelola oleh bendahara asrama serta dapat diakses oleh seluruh penghuni untuk meningkatkan transparansi dan meminimalisir kesalahan pencatatan yang sering terjadi pada sistem manual. Dengan adanya aplikasi ini, proses administrasi asrama menjadi lebih terstruktur, efisien, dan mudah diakses kapan saja.
